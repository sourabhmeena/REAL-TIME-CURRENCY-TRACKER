Help

  • local host: 127.0.0.1, port:8000, version: v1'
  • Specify the currencies to track. (e.g., USD,EUR,GBP)
    • Route: /fetch-currency
    • base: The base currency (optional) default INR
    • symbol: the currency you want to track with respect to base currency (optional) default USD
    • Enter the three-letter currency code of your preferred base currency.
  • Remove the currency. (e.g., USD,EUR,GBP)
    • Route: /remove-currency
    • currency: add the currency you want to remove
    • Enter the three-letter currency code of your preferred base currency.
  • Convert the currency
    • Route: /convert-currency
    • amount (required): The amount to be converted.
    • from (required): The three-letter currency code of the currency you would like to convert from.
    • to (required): The three-letter currency code of the currency you would like to convert to.
  • Currencies supported by the api
    • Route: /available-currency
  • List all preferred currencies stored by the user
      Route: /display-currency
  • for exiting press Command+C
